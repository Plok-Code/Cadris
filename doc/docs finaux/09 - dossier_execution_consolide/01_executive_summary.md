# Executive summary

## Produit

Cadris aide a transformer un projet SaaS encore flou en dossier d'execution exploitable via une mission documentaire multi-agents.

## Cible

Solo founders et petites equipes qui n'ont pas une structure inter-metier deja solide.

## MVP

Le MVP est la tranche verticale `Demarrage` :
- projet
- mission
- question utile
- reponse
- premier artefact
- dossier markdown

## Stack

- web Next.js
- control-plane FastAPI
- runtime Python
- renderer separe
- Postgres canonique
- Restate execution

## Risques principaux

- derive de scope
- confusion entre chat et canonique
- auth et autorisation insuffisantes
- UI trop complexe trop tot
