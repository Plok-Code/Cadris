# Feature specifications

## F-01 Projects

Le systeme doit permettre :
- lister les projets
- creer un projet
- voir son statut mission courant
- reprendre une mission active

## F-02 Mission creation

Le systeme doit permettre :
- ouvrir une mission `Demarrage`
- stocker un intake libre texte
- lancer le runtime

## F-03 Mission read model

La vue mission doit afficher :
- titre
- statut mission
- synthese
- question active si presente
- artefacts en cours
- prochain pas
- progression minimale

## F-04 User answer

Le systeme doit permettre :
- saisir une reponse texte
- valider la reponse
- relancer la mission

## F-05 Dossier

Le systeme doit permettre :
- lire un dossier markdown
- voir les sections
- voir un indicateur de qualite simple

## F-06 Errors and empty states

Le systeme doit expliciter :
- mission introuvable
- dossier non disponible
- chargement
- action impossible
