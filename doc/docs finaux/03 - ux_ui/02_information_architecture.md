# Information architecture

## Navigation principale

- `Mes projets`
- `Mission`
- `Dossier`
- `Revision` plus tard
- `Parametres` plus tard

## Ecrans MVP

### Mes projets
- liste des projets
- creation de projet
- creation ou reprise de mission

### Mission
- contexte de mission
- synthese superviseur
- question active
- artefacts en cours
- progression

### Dossier
- lecture principale
- sections
- qualite

## Logique de hierarchy

- `Mes projets` = entree de travail
- `Mission` = travail actif
- `Dossier` = sortie canonique rendue
