# Screen contracts

## Mes projets

### But
- creer ou reprendre un projet

### Elements obligatoires
- liste projets
- bouton creation
- statut mission
- CTA reprise ou ouverture de mission

## Mission

### But
- comprendre ou en est la mission
- traiter la question active

### Elements obligatoires
- titre mission
- statut
- synthese
- prochain pas
- question active si `waiting_user`
- artefacts
- progression minimale

## Dossier

### But
- lire le premier dossier genere depuis le canonique

### Elements obligatoires
- titre dossier
- resume
- qualite
- sections
- markdown lisible
