# Design system and accessibility

## Tokens structurants

- fond principal : `#FBFAF7`
- surface : `#E7E0D5`
- bordure : `#CDC3B3`
- texte fort : `#2E3431`
- accent : `#3F7277`

## Typographies

- interface : `Public Sans`
- mono systeme : `IBM Plex Mono`

## Composants prioritaires

- `AppShell`
- `ProjectListItem`
- `MissionContextBar`
- `MissionSummaryCard`
- `QuestionCard`
- `ArtifactBlockCard`
- `DossierSection`
- `StatusTag`
- `InlineNotice`
- `EmptyState`

## Accessibilite minimale

- contrastes suffisants
- focus visible
- labels explicites
- messages d'erreur textuels
- cibles interactives >= `44px`
- aucune information critique porte par la seule couleur
