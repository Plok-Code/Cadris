# Stack and ADRs

## Stack imposee

- Next.js 15 + React 19 + TypeScript
- FastAPI + Pydantic v2
- Python 3.13
- OpenAI Agents SDK
- Restate
- PostgreSQL
- S3
- SSE
- markdown first, PDF later

## ADRs closes

### ADR-001
Choix d'une architecture polyglotte plutot qu'un monolithe Next.js.

### ADR-002
Choix de `PostgreSQL` comme verite metier canonique.

### ADR-003
Choix de `Restate` pour l'execution durable.

### ADR-004
Choix d'une premiere tranche verticale `Demarrage` avant les extensions.

### ADR-005
Choix d'un design `mission-first`, sobre, editorial et structurel.
