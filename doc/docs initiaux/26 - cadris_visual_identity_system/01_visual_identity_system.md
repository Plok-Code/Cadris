# 01_visual_identity_system

## Vue d'ensemble du systeme visuel

Le systeme visuel retenu pour Cadris est :
**editorial, structurel, produit-compatible et expert accessible**.

Il doit faire sentir :
- de la clarte ;
- de la tenue ;
- de la precision ;
- une autorite calme ;
- une intelligence visible sans theatre.

Le systeme ne cherche pas a "faire IA".
Il cherche a montrer qu'un projet devient plus lisible, plus stable et plus transmissible.

## Principes directeurs

### 1. Lisibilite avant effet
La marque doit rester tres facile a lire dans :
- une landing ;
- une interface de travail ;
- un dossier exporte ;
- des ecrans denses avec statuts et niveaux de certitude.

### 2. Structure visible
La promesse "cadre clair" doit se traduire visuellement par :
- des grilles ;
- des cadres ;
- des reperes ;
- des couches ;
- des sections nettes ;
- des statuts bien distingues.

### 3. Expert accessible
Le systeme ne doit etre ni froid, ni elitiste, ni professoral.
Il doit sembler :
- competent ;
- precis ;
- sobre ;
- utilisable ;
- accueillant sans etre familier.

### 4. Continuite marque / produit
Le systeme est pense pour servir :
- l'acquisition ;
- la lecture du produit ;
- la consultation du dossier ;
- les ecrans de progression, de blocage et de validation.

### 5. Retenue des accents
La personnalite vient de la composition, du rythme, du contraste et des formes.
Elle ne vient pas d'une multiplication de couleurs, d'effets ou d'illustrations spectaculaires.

## Cohherence avec la strategie visuelle

Le systeme reprend directement les choix du dossier 25 :
- direction : **Architecture editoriale calme** ;
- expression : **expert accessible** plutot que premium demonstratif ;
- territoire principal : editorial + systeme ;
- preuve produit : artefacts, schemas, etats, annotations ;
- humain : presence rare et discrete ;
- palette retenue : **Mineral petrole** ;
- objectif : servir a la fois la marque et le produit des le debut.

## Architecture du systeme

Le systeme visuel repose sur 5 briques stables :

### 1. Palette
- base minerale claire ;
- accent principal petrole / bleu-vert maitrise ;
- couleurs de statut distinctes mais contenues ;
- contrastes suffisamment nets pour ecrans denses et dossiers longs.

### 2. Typographie
- une sans serif principale pour titres, texte et interface ;
- une mono secondaire pour annotations, statuts, donnees et references ;
- hierarchie nette, peu de styles, peu d'effets.

### 3. Langage graphique
- cadres ;
- filets ;
- grilles ;
- onglets ;
- marqueurs ;
- couches ;
- connecteurs simples.

### 4. Iconographie
- monoline geometrique ;
- precise ;
- non ludique ;
- utile avant tout.

### 5. Imagerie
- artefacts, interfaces, schemas, compositions abstraites construites ;
- humain rare et contextuel ;
- jamais de robot, de visage heroique ou de scene corporate.

## Regles globales d'usage

### Composition
- travailler d'abord en zones claires et plans de lecture nets ;
- privilegier une mise en page ancree plutot que des elements flottants ;
- utiliser l'asymetrie avec discipline, jamais pour dramatiser artificiellement.

### Couleur
- la base neutre doit representer la majorite du systeme ;
- l'accent principal sert la navigation, l'activation et les points de focus ;
- les couleurs de statut ne remplacent jamais les labels textuels.

### Typographie
- une page ou un ecran ne doit pas donner l'impression de changer de voix selon les sections ;
- la graisse et la taille font la hierarchie ;
- les effets de casse, d'italique ou de largeur doivent rester rares.

### Densite
- Cadris peut montrer de l'information, mais jamais comme un cockpit opaque ;
- les ecrans denses doivent garder des respirations, des sous-titres et des separations franches.

### Illustration
- chaque image doit expliquer quelque chose : structure, progression, statut, coherence, transmission ;
- aucune image purement decorative ne doit devenir centrale.

## Regles d'interdiction

Le systeme ne doit pas :
- utiliser des halos neon, spheres, galaxies, particules ou codes "IA magique" ;
- tomber dans le bleu corporate pur, le luxe statutaire ou le cyber noir-vert ;
- multiplier les accents, textures et effets jusqu'a brouiller la lecture ;
- anthropomorphiser les agents sous forme de mascottes, robots ou personnages ;
- rendre les bloquants, inconnus et hypotheses differenciables par couleur seule ;
- dependre d'un symbole logo pour exister.

## Decision de travail

Le systeme visuel Cadris est un systeme :
**clair, mineral, structurel, typographique et produit-compatible**, concu pour inspirer confiance par la lisibilite et par la maitrise visible du projet.
