# 08_value_offer_research_prompts

## Prompt 1 — Structures d’offre comparables
Analyse comment des produits comparables structurent une offre qui n’est ni un app builder pur, ni une simple documentation, ni un service de conseil. Je veux voir comment ils présentent leur livrable principal, leur valeur récurrente éventuelle, leur point d’entrée, leurs limites et leur lecture produit. Sors des patterns de structure d’offre compréhensibles pour un builder post-proto.

## Prompt 2 — Promesses de valeur dominantes
Observe les promesses de valeur dominantes chez les outils proches du build assisté par IA, de la source de vérité produit, de la mémoire de projet et de la continuité de développement. Je veux identifier les bénéfices les plus mis en avant, ceux qui reviennent partout, et ceux qui semblent encore crédibles ou distinctifs pour une offre comme Cadris.

## Prompt 3 — Bénéfices saturés ou banalisés
Repère les bénéfices qui sont devenus banals ou sur-utilisés dans cet espace : vitesse, automatisation, “build faster”, “ship faster”, “from idea to app”, documentation automatique, copilote total, etc. Je veux comprendre quels bénéfices éviter comme promesse principale pour ne pas faire entrer Cadris dans un terrain saturé.

## Prompt 4 — Bundles d’offre plausibles
Cherche quels bundles d’offre paraissent les plus plausibles pour un produit comme Cadris :
- artefact concret seul ;
- continuité légère seule ;
- hybride artefact + continuité ;
- projet actif comme unité de valeur ;
- logique ponctuelle vs récurrente bornée.  
Je veux voir les avantages, les risques de perception et les compromis de chaque bundle.

## Prompt 5 — Risques de flou dans la proposition de valeur
Analyse tous les endroits où une offre comme Cadris peut devenir floue dans la tête d’un prospect. Je veux savoir à quel moment elle peut être confondue avec :
- un app builder ;
- un générateur de doc ;
- un outil de gestion de projet ;
- un service de conseil ;
- un copilote de build autonome.  
Pour chaque risque, propose les formulations qui clarifient le plus vite la valeur réelle de l’offre.
