# 03_creative_direction

## Direction recommandee

La direction creative principale recommandee est :
**Architecture editoriale calme**.

## Pourquoi cette direction est retenue

Elle est la plus coherente avec ce que le projet confirme deja :
- Cadris est structurante, fiable, claire et lucide ;
- la promesse de facade est le cadre clair, pas la magie ;
- la marque doit etre logicielle, pas cabinet ;
- le produit doit montrer des preuves, pas des effets ;
- l'univers doit rester compatible avec une interface reelle.

Cette direction permet de tenir la bonne tension :
- assez sobre pour etre credible ;
- assez distincte pour ne pas devenir banale ;
- assez produit pour eviter la lecture "simple doc" ;
- assez calme pour rester anti-hype.

## Codes visuels a privilegier

### Composition
- grilles visibles mais non rigides ;
- blocs bien cadres ;
- colonnes, marges et respirations volontaires ;
- alternance de plans larges et de details d'artefacts ;
- usage de lignes de structure, cadres, reperes et separateurs.

### Typographie
- une sans serif precise et stable comme base ;
- une hierarchie editoriale nette ;
- un usage possible d'une mono tres moderee pour annotations, etats ou donnees ;
- peu d'effets typographiques gratuits.

### Palette
- base minerale et sobre ;
- couleurs de fond claires ou legerement teintees ;
- noirs, graphites, encres, pierres, sables, craies ;
- un accent principal froid-maitrise de type petrole, sauge profonde ou bleu-vert contenu ;
- eventuellement un accent secondaire ponctuel pour l'alerte ou la priorite.

### Formes et motifs
- rectangles, cadres, couches, trames, filets, marqueurs d'etat ;
- motifs d'assemblage, d'annotation ou de liaison ;
- geometrie simple, nette, stable ;
- absence de blobs, d'orbes, de fumee ou de metal liquide.

### Iconographie et illustration
- schemas simples ;
- cartes de blocs ;
- vues de systeme ;
- details de documents ou d'interfaces ;
- illustrations abstraites construites, jamais fantaisistes.

### Motion
- reveals progressifs ;
- alignement ;
- mise en ordre ;
- apparition par couches ;
- transitions qui suggerent clarification et stabilisation.

## Codes a eviter

- gradients neon ou galaxie IA ;
- 3D demonstrative ;
- symbolique "cerveau / etoiles / magie / autonomie totale" ;
- ambiance hacker noire-verte ;
- premium luxe avec dorures, serif demonstrative ou materiaux trop statutaires ;
- univers pastel gentil ou startup playful a mascottes ;
- surdensite de tableaux, labels et signaux qui ferait "outil d'audit" ou "gouvernance enterprise".

## Expression attendue

Si la direction est bien traduite, Cadris doit paraitre :
- rigoureux sans etre raide ;
- calme sans etre froid ;
- premium sans etre exclusif ;
- technique sans etre dominateur ;
- distinctif sans etre a la mode.

## Recommendation operative pour la suite

Le GPT 26 devra traduire cette direction en systeme visuel concret autour de cinq axes :
- typographie ;
- palette ;
- grammaire de mise en page ;
- iconographie / motifs ;
- regles de contraste entre clarte, etat, blocage et validation.
