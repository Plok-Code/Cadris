# 06_blocking_questions

## Questions bloquantes restantes

### 1. Quelle est la matrice minimale de documents obligatoires par type de mission ?
**Pourquoi elle bloque :**
Le produit promet un corpus documentaire large, mais un dossier d'execution ne peut pas etre juge serieusement sans savoir quels livrables sont obligatoires, optionnels ou hors scope selon le contexte.

**Ce qu'il faut obtenir pour avancer :**
Une grille simple du type :
- obligatoire ;
- recommande ;
- optionnel ;
- hors scope.

### 2. Quelle est la doctrine de visibilite des echanges inter-agents pour l'utilisateur ?
**Pourquoi elle bloque :**
Le coeur de valeur repose sur une equipe d'agents qui se voit et se challenge. Il faut maintenant trancher comment cette cooperation est exposee sans rendre l'interface illisible.

**Ce qu'il faut obtenir pour avancer :**
Une doctrine claire sur :
- ce qui est visible en direct ;
- ce qui est visible sous forme de synthese ;
- ce qui reste purement trace / audit.

### 3. Quel est le seuil de qualite pour considerer un document "suffisamment solide" ?
**Pourquoi elle bloque :**
Le systeme produit des artefacts, pas seulement des brouillons. Sans seuil de qualite, on ne sait pas quand un agent doit continuer, demander une validation ou laisser une reserve.

**Ce qu'il faut obtenir pour avancer :**
Une grille courte par document :
- brouillon ;
- suffisant pour discussion ;
- suffisant pour decision ;
- valide avec reserves.

### 4. Quels agents sont vraiment obligatoires au MVP ?
**Pourquoi elle bloque :**
Le modele cible peut vite devenir trop large. Il faut figer le noyau de specialites sans lequel le produit perdrait sa promesse, et distinguer les agents optionnels.

**Ce qu'il faut obtenir pour avancer :**
Une liste tranchee :
- superviseur obligatoire ;
- agents coeur obligatoires ;
- agents conditionnels ;
- agents V2.

## Priorite recommandee
1. matrice minimale des documents obligatoires ;
2. doctrine de visibilite des echanges inter-agents ;
3. seuil de qualite des livrables ;
4. roster exact des agents obligatoires.
